# 🎯 Fundraising Intern Portal  

A web-based fundraising platform where interns can share their referral codes, receive donations in multiple currencies, and track their progress through a dashboard and leaderboard.  

---

## 🚀 Features  
✅ **User Dashboard** – Shows referral code, total amount raised (with currency conversion), and transaction history.  
✅ **Leaderboard** – Displays top fundraisers ranked by total INR raised (with original currency breakdown).  
✅ **Multi-Currency Donations** – Supports INR, USD, EUR, GBP, JPY with real-time conversion to INR.  
✅ **Transaction History** – Users can view detailed donation records.  
✅ **Logout & Authentication** – Secure login/logout flow.  

---

## 🛠️ Tech Stack  
- **Frontend**: React.js  
- **Backend**: Node.js + Express  
- **Database**: MongoDB (Atlas)  
- **Styling**: CSS  
- **Hosting**: (To be updated if deployed)
